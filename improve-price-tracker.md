# IMPROVE — Competitor price tracker

**Input:**

> I want to build a tool for small e-commerce sellers that tracks competitor
> price changes automatically and suggests when to adjust their own pricing.

Scores: 4 / 4 / 2. Real problem, but automatic price tracking means live
scraping — the thing that makes it unbuildable in a week.

The test for this branch: does the MVP still contain automatic tracking? It
doesn't. Manual input in v1, scraping moved to Future Version. That scope
reduction is the entire reason IMPROVE exists.

---

## 1. Verdict

IMPROVE

## 2. Target User

Small e-commerce sellers who need to adjust their prices competitively.

## 3. Core Problem

Small sellers struggle to keep track of competitor price changes, affecting their
ability to price their products effectively.

## 4. Evaluation Scores

- **User Clarity Score:** 4 — The target user is well-defined, making it clear who
  the tool is for.
- **Problem Strength Score:** 4 — There is a significant pain point for small
  sellers in tracking competitor prices, impacting their competitiveness.
- **MVP Feasibility Score:** 2 — The idea requires external data for competitor
  tracking, making it difficult for a beginner to build within 7 days.

## 5. Why It Might Work

If simplified, this tool could provide a valuable service to small sellers,
helping them adjust prices based on competitor data.

## 6. Main Risks

- Difficulty in sourcing reliable competitor pricing data.
- The complexity of implementing automatic tracking for a beginner.
- Potential for saturation in the market with similar tools.

## 7. Revised Idea

Create a simple tool that manually allows small e-commerce sellers to input
competitor prices and receive basic suggestions for their own pricing adjustments
based on those inputs.

## 8. MVP Features

1. Manual input for competitor prices.
2. Basic pricing adjustment suggestions based on competitor data.
3. User-friendly interface to manage inputted prices.

## 9. 7-Day Building Plan

- **Day 1:** Research user needs and create wireframes.
- **Day 2:** Set up a basic UI layout.
- **Day 3:** Implement manual input functionality for competitor prices.
- **Day 4:** Create a simple algorithm for pricing suggestions.
- **Day 5:** Integrate the UI with the pricing suggestion feature.
- **Day 6:** Test the tool with potential users and gather feedback.
- **Day 7:** Refine the tool based on feedback and prepare for launch.

## 10. Future Version

Incorporate automatic tracking of competitor prices, real-time data updates, and
payment integration for a more comprehensive solution.

## 11. X/Twitter Launch Post

Launching a tool for small e-commerce sellers to track competitor prices manually
and get pricing suggestions! Simple, effective, and built for your needs. Stay
competitive!
